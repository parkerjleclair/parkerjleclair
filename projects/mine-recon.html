Mine Reconciliation Using UAV Photogrammetry & R

Summary

This project uses monthly UAV photogrammetry surveys to generate high- resolution DSMs, remove outliers, compute lowest-elevation surfaces across all surveys, and compare modeled versus actual ore and waste extraction volumes. The workflow integrates SiteScan, CloudCompare, R, and ArcGIS Pro to support mine planning and geologic interpretation.

Methods Overview

  • UAV flight planning and photogrammetry processing
  • Point cloud classification and outlier removal
  • DSM generation and grid-based downsampling
  • Lowest-elevation surface extraction across all survey dates
  • Comparison to geologic block model using R and GIS
  • Volume change calculations for ore/waste reconciliation
